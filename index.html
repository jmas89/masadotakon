Hello this is our presentation on how to change colors, font face and Sizes of Word or Text

The Text Color

Setting the Color of a text, can be done easily.

To change the Color of the text you must type the fallowing syntax:

Example
< p style="color:red;"> kalbo ni joel 123
< p style="color:blue;"> kalbo ni joel 123
< p style="color:yellow;"> kalbo ni joel 123

The Text Font

Setting the Font of a text, can be done by picking unique fonts or searching in google.

To change the Font of the text you must type the fallowing syntax:

Example
< p style="font-family:Edwardian Script ITC;"> kalbo ni joel 123
< p style="font-family:Curlz MT;"> kalbo ni joel 123
< p style="font-family:cooper;"> kalbo ni joel 123

The Text Font

Setting the Size of a text, can by changing the percentage of the numbers you can only pick 1-10

To change the Size of the text you must type the fallowing syntax:

Example
< font size='2'> kalbo ni joel 123
< font size='5'> kalbo ni joel 123
< font size='10'> kalbo ni joel 123
